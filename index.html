  • Prompt: do something with color.


    Intent: use colors to increase the intelligibility of the harmonic complexity of a musical arrangement.


    Process:

    1. 1. I arranged and multi-track recorded an a capella version of the song 'Born a Worm' by Dan Reeder.
    2. 2. I converted each voice recording into represetative MIDI data (a digital piano roll, if you will) using Ableton Live's native audio-to-MIDI conversion. A good degree of manual correction was required for each part, but after doing so I ended up with a software representation like this for each voice.
    3. 3. I modified a version of this script to import the MIDI representation of each voice into After Effects, which created keyframes in-time with the recording for the pitch, velocity (intensity), and duration of each note. Here is what that looked like in After Effects, and here is the modified script I used; for some reason the original script didn't run on my Mac.
    4. 4. I assigned the hue parameter for the box representing each voice so that every ascending note (C, C#, D, etc) corresponded to a 30 degree rotation on the color wheel, so every C was the same color, every F# was the same color, etc., for all notes. I assigned the velocity parameter for each voice to a scale multiplier of its representative box, so that box size responded dynamically to the loudness of each note. The specific scripting I wrote to correspond note pitch and velocity to box color and size is pictured here.
